OS X下搭建Hexo

发表于 2015-04-02   |  

  • 首先，介绍一下Hexo，官网介绍如下:

    A fast, simple & powerful blog framework

    Hexo是一个基于 Node.js 的静态博客程序,作者是台湾大学生tommy351，其编译上百篇文字只需要几秒。hexo生成的静态网页可以直接放到GitHub Pages，BAE，SAE等平台上。

    P.S：Hexo有自己的官方教程，英文版的，但不难阅读。

你得有一个github账户

已有的直接登录，没有的直接sign up
然后下载XCcode(AppStore里，翻墙下载会快很多)、git吧

关于下载git for Mac

请先下载好Xcode,然后进入git for Mac的网址，download，相关教程网上已有，这里给一个教程。

建立一个repo

登陆你的github，“＋”处creat a repository，名字为username(你自己的用户名).github.io,其他设置可以不用调整。然后在你的Popular repositories处会有一个repo出现。

下载一个Subline Text2
